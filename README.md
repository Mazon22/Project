# filecreate

Консольная программа для создания текстовых файлов с вводом содержимого и выбором пути сохранения.

## Возможности

- Меню выбора действий
- Ввод нескольких строк текста
- Указание пути для сохранения файла
- Цикличный возврат в меню после выполнения операции

## Запуск

```
python filecreate.py
```

## Как работает

1. При запуске появляется меню
2. Выбираешь «Создать файл» (пункт 1)
3. Вводишь строки — пустая строка завершает ввод
4. Указываешь путь, куда сохранить файл
5. Файл создан, снова показывается меню
